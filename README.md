# Clouds classification app

A web app for predicting clouds type based on photographs.

## Technologies
* Angular
* Bootstrap

## How to run

> Prerequisites:
> * Node installed.
> * [Server](https://github.com/Rys-Nowak/clouds-app-server) started.

```bash
npm install
ng serve
```

The app will start at `http://localhost:4200/`.
